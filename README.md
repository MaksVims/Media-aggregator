🎬 Агрегатор Медиаконтента
Веб-приложение для поиска информации о кинофильмах и художественных произведениях с возможностью перехода на внешние платформы для легального просмотра.
🚀 О проекте
Приложение позволяет пользователям изучать подробную информацию о фильмах, актерском составе и рецензиях. Для зарегистрированных пользователей доступен расширенный функционал по управлению личной коллекцией.
Ключевые возможности:
Умный поиск: фильтрация по названию, году выпуска и рейтингу.
Детальные страницы: полная информация о фильме, включая постеры, описание и ссылки на стриминги.
Профили актеров: кликабельный список создателей с переходом на страницу биографии и фильмографии.
Система закладок: авторизованные пользователи могут формировать собственную коллекцию медиаконтента.
Адаптивность: интерфейс полностью оптимизирован под различные разрешения экранов.
🛠 Технологический стек
Frontend
Core: Next.js (React, TypeScript)
Styling: Tailwind CSS
State Management: MobX
Backend & API
Backend as a Service: Firebase (Auth, Firestore)
Data API: Kinopoisk API Unofficial
🏗 Архитектура и реализация
Авторизация и безопасность
Механизм: Реализован на базе Firebase Auth. При входе генерируется токен, сохраняемый в cookie (время жизни — 1 час).
Context API: Состояние авторизации управляется через src/contexts/AuthContext.tsx. Это позволяет эффективно пробрасывать данные пользователя по дереву компонентов без избыточных запросов к БД.
Сессии: При обновлении страницы или открытии новой вкладки приложение проверяет наличие куки и автоматически восстанавливает сессию пользователя.
Взаимодействие с данными
Логика разделена на специализированные сервисы для обеспечения чистоты кода:
src/api/MovieService.ts — получение данных о фильмах.
src/api/StaffService.ts — работа с данными об актерах и съемочной группе.
src/api/FirebaseAuthService.ts — CRUD операции пользователей и логика входа.
src/api/FirebaseCollectionService.ts — работа с коллекциями и закладками пользователей.
service/firebase.ts — конфигурация и инициализация Firebase SDK.
