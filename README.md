# 🎬 Агрегатор Медиаконтента

Веб-приложение для поиска информации о кинофильмах и художественных произведениях с возможностью перехода на внешние платформы для легального просмотра.

## 🚀 О проекте

Приложение предоставляет пользователям удобный интерфейс для изучения кинематографа. Основной акцент сделан на интеграции с внешними API и предоставлении актуальной информации о фильмах, актерах и рецензиях.

### Ключевые возможности:
*   **Умный поиск:** Поиск по названию и фильтрация по году выпуска и рейтингу.
*   **Персональные коллекции:** Авторизованные пользователи могут добавлять фильмы в «Закладки».
*   **Интерактивность:** Кликабельные профили актеров с полной фильмографией и фактами.
*   **Адаптивность:** Полная поддержка мобильных устройств и планшетов (Responsive Design).
*   **Легальный просмотр:** Прямые ссылки на внешние стриминговые платформы.

## 🏗 Архитектура и реализация

### Авторизация и сессии
- **Механизм:** Реализован в `src/contexts/AuthContext.tsx` с использованием React Context для эффективного проброса состояния пользователя.
- **Безопасность:** При авторизации выдается токен, который сохраняется в **Cookie** (время жизни — 1 час). 
- **Persistence:** При обновлении страницы или открытии новой вкладки приложение проверяет куки и восстанавливает сессию, отправляя запрос в базу данных.

### Структура API и сервисов
Логика взаимодействия с данными разделена по зонам ответственности:
- `service/firebase.ts` — Базовая конфигурация Firebase.
- `src/api/FirebaseAuthService.ts` — Регистрация и аутентификация.
- `src/api/FirebaseCollectionService.ts` — CRUD операции с пользовательскими коллекциями.
- `src/api/MovieService.ts` — Методы получения данных о фильмах из внешнего API.
- `src/api/StaffService.ts` — Получение данных об актерах и съемочной группе.
